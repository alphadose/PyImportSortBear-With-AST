# PyImportSortBear-With-AST

## How to use :-
```
python3 traverse.py <file_name>
```

## Example :-
```
python3 traverse.py sample.py
```

## Output :-
```
Package b derived from package a
Parent package: a
Package d derived from package c
Parent package: c
Parent package: e
Package f aliased as g
Parent package: f
Package h aliased as i
Parent package: h
```
